### Proyectos Destacados en los que he trabajado

#### Pipeline de Datos Industrial (Arquitectura Medallion)
* **Descripción:** Diseñé e implementé un pipeline de datos integral para una planta de manufactura de pinturas (**Pintugama**). El sistema automatiza la migración y transformación de datos críticos desde el ERP Odoo hacia un Data Warehouse centralizado en PostgreSQL.
* **Detalles técnicos:** Estructurado bajo la arquitectura Medallion en tres niveles:
    * **Bronze:** Ingesta de datos crudos (Raw).
    * **Silver:** Limpieza, tipado y validación de esquemas con Python y Pandas.
    * **Gold:** Modelado de datos agregado y cálculo de KPIs clave para la toma de decisiones de la directiva.

#### Servidor de Monitoreo e Infraestructura Local
* **Descripción:** Configuración y despliegue de un nodo de servicios y monitoreo centralizado utilizando un Mini PC en entorno de producción.
* **Detalles técnicos:** Implementación sobre **Ubuntu Server 24.04**, gestión de servicios contenedorizados con **Docker**, enrutamiento seguro mediante proxy inverso con **Nginx** y optimización de accesos seguros mediante **SSH** y **Mosh**.
